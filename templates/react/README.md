# React Template

This template demonstrates how to build React applications with FHEVM SDK integration.

## Available Examples

### Private Parking React
**Location**: `../../examples/private-parking-react/`

A modern React application built with Vite showcasing FHEVM SDK:
- React 18 with TypeScript
- Vite for fast development
- FHEVM SDK hooks integration
- Wallet context management
- Tailwind CSS styling

**Quick Start**:
```bash
cd ../../examples/private-parking-react
npm install
npm run dev
```

## Features

- ✅ React 18 with Hooks
- ✅ TypeScript support
- ✅ FHEVM SDK integration
- ✅ Vite for fast builds
- ✅ Modern component architecture
- ✅ Tailwind CSS styling

## Getting Started

1. Copy the example to your project
2. Install dependencies: `npm install`
3. Configure your contract addresses
4. Start development: `npm run dev`
